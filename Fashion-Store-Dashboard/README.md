Fashion Store Analytics Dashboard

Tools Used: Qlik Sense

Overview
This dashboard analyzes sales performance, customer behavior, and promotion effectiveness for a fashion retail business.

Key Insights
• Revenue performance and order value trends
• Top products and category performance
• Customer purchase behavior and geographic revenue distribution
• Channel contribution and sales drivers
• Impact of discounts and promotions on revenue

Data Preparation
• Loaded structured sales datasets into Qlik Sense
• Performed data cleaning and field associations
• Built data model using Qlik load editor

Features
• Interactive filters for date, category, and channel
• KPI indicators for business performance tracking
• Visual insights for decision-making
